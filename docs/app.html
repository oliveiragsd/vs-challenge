get /products

Requisição para listar produtos

Secured by jwtAuth

JWT token based authentication

  • Request
  • Response
  • Security

Query Parameters

  • name: (string)

    Nome do produto

    Example:

    seringa
  • brand: (string)

    Marca do produto

    Example:

    BUNZL
  • fromPrice: (number)

    Preço mínimo do produto

    Example:

    19.9
  • toPrice: (number)

    Preço máximo do produto

    Example:

    100
  • fromStock: (number)

    Estoque mínimo do produto

    Example:

    0
  • toStock: (number)

    Estoque máximo do produto

    Example:

    10
  • sort: (string)

    Ordenação da lista. Parâmetro a ser utilizado e sentido (ascendente - asc ou descendente - desc) separados por dois pontos (:)

    Example:

    price:desc
  • perPage: (number)

    Quantidade de itens exibidos por página

    Example:

    50
  • page: (number - default: 1)

    Página atual da listagem

    Example:

    2
